# Images Directory

This directory is for storing all campsite images that will be displayed on the website.

## Adding Images

1. Add your images to this directory
2. Use descriptive filenames (e.g., `waterfall-morning-mist.jpg`, `cabin-exterior.jpg`)
3. Reference images in data files as `/images/filename.jpg`

## Recommended Images

Based on the website content, you'll want to add:

### Essential Images
- **Hero/Banner image**: A stunning main image for the home page
- **Waterfall images**: Photos of all three waterfalls
- **Accommodation photos**:
  - Tent sites
  - Powered caravan/RV sites
  - Camping cabins (interior and exterior)
  - Group camping areas

### Nice to Have
- Hiking trails
- Campfire scenes
- Wildlife (if available)
- Facilities (bathrooms, camp kitchen, etc.)
- Seasonal photos (different times of year)
- Action shots (swimming, hiking, etc.)

## Image Optimization Tips

Before adding images:
1. **Resize** images to appropriate dimensions (max 1920px wide for hero images, 800px for gallery images)
2. **Compress** images using tools like:
   - [TinyPNG](https://tinypng.com/)
   - [Squoosh](https://squoosh.app/)
   - [ImageOptim](https://imageoptim.com/) (Mac)
3. **Format**: Use JPG for photos, PNG for graphics/logos
4. Consider **WebP** format for better compression (supported by modern browsers)

## File Naming Convention

Use lowercase with hyphens:
- ✅ `waterfall-morning-mist.jpg`
- ✅ `cabin-exterior-01.jpg`
- ❌ `Waterfall Morning Mist.jpg`
- ❌ `cabin_exterior_01.jpg`
