# Directions Images

This folder contains images for the "How to Get Here" page showing the route to the campsite.

## Expected Images

Based on the HowToGetHereView.vue component, these images should be added:

1. `from-kampala.jpg` - Route from Kampala (highway views, road signs)
2. `mbale-town.jpg` - Mbale town area / arrival point
3. `final-approach.jpg` - Final road to campsite (unpaved road, signage)

## Image Guidelines

- **Format**: JPG or PNG
- **Resolution**: 1200-1600px wide recommended
- **Content**: Clear photos showing the route, road conditions, landmarks, or signage
- **Naming**: Use the exact filenames listed above for automatic display

## Alternative Images

You can also add:
- Maps or GPS screenshots
- Landmark photos
- Road condition photos during different seasons
- Signage photos

Simply reference the new image paths in the `directionSteps` array in `src/views/HowToGetHereView.vue`.
